Image Carousel Feature

The Carousel page showcases a dynamic image slider built with the Bootstrap 5 Carousel component. This feature allows visitors to scroll through four original photographs with smooth transitions and built-in navigation controls. The carousel adds movement and visual interest to the site, making the user experience more engaging.
